🧠 Core Input

Brain dump input — voice (Whisper API) or free-text
Session start prompt — "What's on your mind today?"
AI processes raw dump into structured data (tasks, worries, noise)


✅ Task Management

Prioritized list — AI ranks tasks by urgency/importance
Kanban board — To Do / In Progress / Done columns, drag & drop
Mind map view — visual node-based layout of tasks and themes
Toggle between Kanban ↔ Mind Map
Mark tasks complete inline (checkbox or swipe on mobile)
Edit tasks — rename, re-prioritize, delete
Add tasks manually after AI generation


📓 Journaling

End Session button — triggers journal flow
AI-generated journal prompt — contextual, based on the dump + completed tasks
Voice or text journal response
AI session summary — brief recap of what you came in with vs. what you accomplished
Mood tag — quick emoji or scale rating at end of session (1–5 or 😩→😊)


📊 Visual Analytics & History

Session history — log of all past sessions
Mood/stress trend chart — over days and weeks (Recharts)
Mental load score — AI-derived score per session, tracked over time
Theme clustering — recurring topics surfaced across sessions (e.g. "work", "finances")
Word cloud — from past brain dumps


📅 Calendar Integration (stretch)

Push tasks to Google Calendar — with AI-suggested due dates/times
Pull existing events — AI factors in your schedule when prioritizing
Conflict detection — flags if your task list is unrealistic given your calendar


🎨 UX & Polish

Minimal, calming UI — designed to reduce anxiety, not add to it
Dark / light mode
Mobile-first responsive design
Onboarding flow — first-time user walkthrough
Auth — Supabase (email or Google login)
Persistent sessions — resume where you left off
